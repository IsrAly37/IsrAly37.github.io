Meus Projetos

Blog Pessoal com Sistema de Publicação

Este projeto permite criar, editar e excluir postagens em um blog, com autenticação de usuários. Oferece um editor de texto rico (WYSIWYG) para formatar as postagens, além de categorias para organizar os conteúdos. O blog inclui uma área administrativa para gerenciar posts e perfil de usuário. O sistema é construído com frontend em HTML, CSS, JavaScript (ou React) e backend com Node.js e banco de dados como MongoDB ou MySQL. O foco está na autenticação segura, gestão de conteúdo e design responsivo.

Assistente Virtual com IA

O Assistente Virtual com IA é um projeto que cria um software capaz de entender comandos de voz ou texto e realizar tarefas automáticas, como marcar compromissos ou enviar mensagens. Utiliza inteligência artificial e processamento de linguagem natural (NLP) para interpretar e responder ao usuário, além de se integrar a serviços como calendários e e-mails. O objetivo é automatizar tarefas diárias, tornando-as mais eficientes e personalizadas através do uso de IA.
